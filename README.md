# 📊 Telco Customer Churn Analysis

## 📌 Project Overview
This project focuses on analyzing customer churn behavior using Exploratory Data Analysis (EDA). The goal is to identify key factors that influence customer churn such as tenure, monthly charges, and total charges.

The analysis helps businesses understand customer retention patterns and improve decision-making strategies.

---

## 🛠️ Tools & Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook

---

## 📊 Key Insights
- Overall churn rate of customers analyzed
- Relationship between tenure and churn identified
- Monthly charges impact on churn studied
- Customer behavior patterns explored through visualizations

---

## 📁 Project Files
- Jupyter Notebook (`.ipynb`) – Complete EDA workflow
- Dataset (`CSV`) – Telco customer dataset
- Visualizations – Charts and graphs generated during analysis

---

## 🎯 Business Objective
To understand why customers leave the service and identify important features that influence churn so that retention strategies can be improved.
